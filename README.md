🎨 [Projeto1_Figma]: Do Figma ao HTML
Este é o meu primeiro projeto prático unindo as etapas de UI Design e Desenvolvimento Front-end. O objetivo foi criar um layout do zero no Figma e realizar a sua implementação fiel utilizando HTML e CSS.

🚀 O que este projeto representa?
Este repositório documenta minha evolução como estudante de ADS, saindo da teoria e aplicando conceitos de prototipagem e marcação de texto.

🛠️ Etapas do Desenvolvimento
Design (Figma): Criação da interface, definição de paleta de cores, tipografia e espaçamentos (Gutter/Grid).

Implementação (HTML/CSS): Tradução dos elementos visuais para código semântico.

Fidelidade Visual: Foco em manter o código o mais próximo possível do design original ("Pixel Perfect").

🧠 O que eu aprendi "do zero":
Como interpretar as propriedades do Figma (Width, Height, Padding, Hex Colors) e passá-las para o CSS.

A importância da semântica no HTML para acessibilidade.

Organização de arquivos e pastas para projetos web.
